# Luxe Dark Salla Theme

Production-ready Salla theme inspired by premium dark fashion storefront experiences.

## Folder Structure

```text
salla-hellstar-theme/
├─ theme.json
├─ layouts/
│  ├─ master.twig
│  ├─ home.twig
│  ├─ collection.twig
│  └─ product.twig
├─ partials/
│  ├─ header.twig
│  ├─ footer.twig
│  └─ product-card.twig
└─ assets/
   ├─ css/theme.css
   └─ js/theme.js
```

## Features

- Dark luxury UI with premium spacing.
- Sticky header and minimal navigation.
- Collection-first responsive product grid.
- Product hover zoom + optional second image swap.
- Lazy-loaded images and subtle reveal animations.
- Reusable components and maintainable structure.

## Local Development

1. Install Salla CLI.
2. Use `salla theme serve` in this directory.
3. Open preview URL and customize from theme settings.
